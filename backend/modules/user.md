# 👤 User Module

## Overview

The User Module manages user accounts, profiles, permissions, and account settings.

---

## Responsibilities

- User Profile
- Account Settings
- Roles & Permissions
- Profile Picture
- Account Status

---

## User Roles

### Admin

- Manage Platform
- Manage Users
- View Analytics

---

### Developer

- Create Projects
- Deploy Applications
- Use AI Assistant
- Manage Smart Contracts

---

### Viewer

- Read-only Access

---

## Profile Information

- Full Name
- Username
- Email
- Bio
- Avatar
- Joined Date

---
