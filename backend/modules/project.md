# 📁 Project Module

## Overview

The Project Module manages software projects within Nexus. It allows developers to create, organize, update, and deploy projects.

---

## Responsibilities

- Create Project
- Update Project
- Delete Project
- Archive Project
- Search Projects
- Manage Team Members

---

## Project Information

- Project Name
- Description
- Owner
- Visibility (Public / Private)
- Status
- Technology Stack
- Created Date
- Updated Date

---

## Status Options

- Draft
- Active
- Completed
- Archived

---

## API Endpoints

GET /projects

GET /projects/:id

POST /projects

PUT /projects/:id

DELETE /projects/:id
