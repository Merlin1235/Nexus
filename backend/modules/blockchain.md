# ⛓️ Blockchain Module

## Overview

The Blockchain Module enables developers to create, deploy, monitor, and interact with smart contracts directly from Nexus.

---

## Responsibilities

- Smart Contract Management
- Wallet Integration
- Network Management
- Contract Deployment
- Transaction Monitoring
- Gas Fee Estimation

---

## Supported Networks

- Ethereum
- Polygon
- Base
- BNB Chain
- Arbitrum
- Optimism

---

## Wallet Support

- MetaMask
- WalletConnect
- Coinbase Wallet

---

## Features

- Deploy Smart Contracts
- Verify Contracts
- Read Contract Data
- Execute Transactions
- View Transaction History
- Monitor Deployment Status

---

## API Endpoints

POST /blockchain/deploy

GET /blockchain/contracts

GET /blockchain/contracts/:id

POST /blockchain/verify

GET /blockchain/networks

GET /blockchain/transactions

---
